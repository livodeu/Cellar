• {pref_network_allow_metered}

    Contrôle si l’application est autorisée à utiliser des connexions réseau marquées comme limitées. Si une connexion est considérée comme mesurée est déterminée à la discrétion du système d'exploitation.

  • {pref_proxy_type}

    Permet la sélection d'un type de proxy (HTTP ou SOCKS). Wikipédia a des informations sur ce qu’est un proxy.

  • {pref_proxy}

    Ce paramètre n'est visible que si un type de proxy a été sélectionné. Entrez l'adresse proxy et, éventuellement, un numéro de port ici.

  • {pref_proxy_restrict}

    Ce paramètre n'est visible que si un type de proxy a été sélectionné. Le serveur proxy peut être limité à être utilisé pour certains hôtes uniquement.

  • {pref_via_vpn}

    Détermine si le logiciel doit utiliser des connexions VPN.

  • {pref_blacklist}

    Définir une liste de blocage. L’application ne chargera aucun fichier à partir d’un hôte figurant sur cette liste. La liste sera mise à jour toutes les {blacklist_update} heures, si possible.

  • {pref_night}

    L'application peut soit suivre les paramètres du système, soit basculer le mode noir à des heures spécifiques de la journée. Ceci est destiné aux appareils plus anciens (avant la version 9). Si l’appareil est en mode économiseur de batterie, il est possible que le mode sombre ne change pas correctement.

  • {pref_clipspy}

    L’application peut observer le presse-papiers du système pour détecter les URL qui pointent vers des ressources téléchargeables. Ce n’est plus possible à partir d’Android 10.

  • {pref_backup_restore}

    Les fichiers téléchargés peuvent être compressés dans un fichier zip. Vous devez sélectionner une application cible pour recevoir le fichier zip. (Les applications ne sont pas autorisées à accéder à la plupart des parties du système de fichiers, telles que le dossier de téléchargements.) Si vous entrez un mot de passe, le contenu du fichier zip sera crypté.

  • {action_systemsettings_app}

    Regardez les paramètres système du {app_name}.

  • {pref_licenses}

    Voir les bibliothèques tierces utilisées par {app_name}.

  • {pref_share_this}

    Exportez l’application en tant que fichier apk qui peut être installé sur d’autres appareils. Notez que l'application ne fonctionnera pas sur chaque appareil Android.
