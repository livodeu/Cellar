• {pref_network_allow_metered}

    Controls whether {app_name} is allowed to use network connections that are tagged as metered. Whether a connection is con­sidered to be metered is deter­mined at the dis­cretion of the operating system. These are usually mobile data connections, like LTE.

  • {pref_proxy_type}

    Allows the selection of a proxy type (HTTP or SOCKS). Wikipedia has infor­mation about what a proxy is.

  • {pref_proxy}

    This setting is visible only if a proxy type has been se­lected. Enter the proxy address and, optionally, a port number here.

  • {pref_proxy_restrict}

    This setting is visible only if a proxy type has been selected. The proxy server can be re­stricted to be used for certain hosts only.

  • {pref_via_vpn}

    Determines whether {app_name} should use VPN connections.

  • {pref_blacklist}

    Set a black­list provider, comparable to a browser's fil­ter list. The app will not load any file from a host given on that list. The list will be up­dated every {blacklist_update} hours, if possible.

  • {pref_night}

    The app may either follow the system settings, or toggle the dark mode at specified hours of day. This is meant for older devices that do not support dark mode natively (Android up to version 9). If the device is in battery saver mode, it is possible that the dark mode does not change correctly.

  • {pref_clipspy}

    This setting is avail­able only up to Android 9. The app can ob­serve the system clip­board to detect URLs that point to down­loadable resources. This is not possible any more from Android 10 on.

  • {pref_backup_restore}

    The downloaded files can be com­pressed into a zip file. You must select a target app to receive the zip file because apps, in­cluding this one, are not allowed to access most parts of the file system, such as the down­loads folder. If you enter a pass­word, the zip file contents will be encrypted.

  • {action_systemsettings_app}

    Have a look at the system settings for {app_name}.

  • {pref_licenses}

    See the third-party libraries used by {app_name}.

  • {pref_share_this}

    Export {app_name} as an apk file that can be in­stalled on other devices. Note that the exported app will not run on every Android device. If your device is fairly new, the app will probably not run on a very old device. {app_name} has also not been designed to run on devices other than phones or tablets.
